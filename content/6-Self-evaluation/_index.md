Throughout my internship at **AWS Vietnam** from **05/04/2026** to **07/27/2026**, 
I had the opportunity to learn, practice, and apply the knowledge gained at university 
into a real working environment.

I participated in the **OCRVID project — an automated video processing application 
using OCR, translation, and dubbing**, where I was responsible for the **AWS 
infrastructure**, including: deploying the database on Amazon RDS MySQL, storing 
video files and subtitles on Amazon S3, deploying the backend on Amazon EC2 using 
Docker, configuring IAM and Security Groups, and connecting all AWS services together 
to ensure stable system operation.

Through this experience, I improved my skills in: **AWS cloud infrastructure 
deployment, IAM/Security Group security configuration, working with S3 and RDS in 
a real-world environment, using Docker to containerize applications, teamwork and 
effective task delegation**.

In terms of work attitude, I always strived to complete assigned tasks well, comply 
with regulations, and actively communicated with team members to ensure project 
progress.
