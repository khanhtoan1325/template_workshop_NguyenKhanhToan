### [Blog 3 - Modernizing applications with Strands Agents and Amazon Bedrock AgentCore](3.3-Blog3/)
This article discusses how AWS leverages Generative AI Agents to automate application modernization at scale. Instead of developers manually analyzing repositories, checking dependencies, upgrading runtimes, migrating SDKs, or refactoring frameworks, this solution employs Strands Agents, AWS Transform Custom, and Amazon Bedrock AgentCore. Together, these components build an AI agent system capable of analyzing codebases, identifying appropriate transformations, generating new transformations as needed, and executing these changes across multiple repositories via AWS Batch. Consequently, enterprises can reduce the time and effort required for application modernization, ensure greater consistency across teams, reuse transformations across projects, and more effectively scale the process of migrating applications to the cloud.
